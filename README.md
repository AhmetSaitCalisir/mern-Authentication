# mern-Authentication

Single page authentication using JWT with Mongodb, Express.js, React.js and Node.js

## Dependencies

# Server

- bcryptjs
- cors
- dotenv
- express
- jsonwebtoken
- mongoose
- nodemon

# Client

- axios
- bootstrap
- jquery
- popper.js
- react
- react-router-dom

## Installation

### For installation of server:

```bash
cd server
npm i
```

### For installation of client:

```bash
cd client
npm i
```

## Initializing App

### For Initializing server:

```bash
cd server
npm run server
```

### For Initializing client:

```bash
cd client
npm run start
```

## config.env

```
PORT = 0000
DB_CONNECTION = connection_string

JWT_SECRET = hexcode
JWT_EXPIRE = 0min
```
